# Flowception: Temporally Expansive Flow Matching for Video Generation

[![Website](https://img.shields.io/badge/Website-Flowception-B31B1B?logo=google-chrome)](https://flowception-meta.github.io/)
[![Paper](https://img.shields.io/badge/Paper-arXiv-B31B1B?logo=arxiv)](https://arxiv.org/abs/2512.11438)

**Flowception** is a non-autoregressive, variable-length video generation framework that interleaves  **discrete frame insertions** with **continuous denoising / flow matching**.
